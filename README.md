# DevOps TP1 - Hardening Linux & CI/CD

## 📋 Description
Projet réalisé dans le cadre du TP1 DevOps : installation et durcissement d'un serveur Linux, 
mise en place d'un dépôt Git avec pipeline CI.

## 🎯 Objectifs
- Installation et hardening d'un serveur Ubuntu Server 24.04
- Configuration d'un dépôt Git avec stratégie de branching GitHub Flow
- Mise en place d'un pipeline CI (lint + tests unitaires)

## 🛠️ Stack technique
- **OS** : Ubuntu Server 24.04 LTS
- **Langage** : Python
- **CI/CD** : GitHub Actions
